WristBoard

Full Keyboard for Apple Watch

About App

WristBoard (previously Shift Keyboard) is a full keyboard for Apple Watch. It supports multiple languages and QWERTY, QWERTZ and AZERTY keyboard layouts with support for autocorrect, text suggestions and custom dictionaries too.

Featured On

WristBoard has been featured on the App Store, 9to5Mac, Cult of Mac, iMore, AppAdvice and iPhon.fr.

Platforms

WristBoard is available on iOS and watchOS here.

Technologies

WristBoard has been built using SwiftUI, Core Data and WidgetKit.
